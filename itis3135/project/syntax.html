Figure 1 shows the correct syntax for a Python program.

  • The first line is a comment. Comments are used to explain what the program does, they are not executed when the program runs and are useful for programmers to understand what the program does.
  • Comments are ignored by the Python interpreter and are preceded by a hash symbol (#).
  • The second line is a variable it is used to store data. In this case, it assigns the value "Hello World!" to the variable my_variable.
  • The third line is a print statement. The print statement displays the text inside the variable my_variable on the screen. The print statement is executed when the program runs.
